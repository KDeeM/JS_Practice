DOM Manipulation

The DOM(Document Object Model) is a JavaScript object representing the format and structure of the HTML content, this means any changes made in the DOM are reflected in the HTML content itself.

The DOM can thus be used to add, alter and remove elements on your page, with the added support of other JavaScript capabilities such as Http Requests, webpages can become dynamic able to even fetch content without the need for page refreshing. Knowledge of the DOM can thus be a powerful tool to learn.


Altering Content
